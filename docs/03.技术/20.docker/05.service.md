## docker 启动命令

-   启动

    ```shell
    systemctl start docker
    ```

## docker 重启命令

-   守护进程重启

    ```shell
    systemctl daemon-reload
    ```

-   重启 docker 服务

    ```shell
    systemctl restart docker
    # or
    service docker restart
    ```

## docker 关闭命令

-   关闭

    ```shell
    docker service docker stop
    # or
    docker systemctl stop docker
    ```
