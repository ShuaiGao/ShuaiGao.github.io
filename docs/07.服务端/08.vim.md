## 2. 正则替换

正则替换分组使用 () 但是需要转义，即 \(\)

```text
# 将文本的
# },
# {
# 替换为 }, {

:%s/},$\n\s\*{/}, {/g
```

例子:

-   删除行尾空格：:%s/\s+$//g

-   删除行首多余空格：%s/^\s*// 或者 %s/^ *//

-   删除沒有內容的空行：%s/^$// 或者 g/^$/d

-   删除包含有空格组成的空行：%s/^\s*$// 或者 g/^\s*$/d

-   删除以空格或 TAB 开头到结尾的空行：%s/^[ |\t]_$// 或者 g/^[ |\t]_$/d

把文中的所有字符串"abc……xyz"替换为"xyz……abc"可以:下列写法

:%s/abc\(._\)xyz/xyz\1abc/g
:%s/\(abc\)\(._\)\(xyz\)/\3\2\1/g
