常用编码

HUP 1 终端断线
INT 2 中断（同 Ctrl + C）
QUIT 3 退出（同 Ctrl + \）
TERM 15 终止
KILL 9 强制终止
CONT 18 继续（与 STOP 相反， fg/bg 命令）
STOP 19 暂停（同 Ctrl + Z）

一个 kill 命令
ps -ef | grep devops | grep 7000 | grep -v grep | awk -F" " '{if($3==1) print $2}' | xargs kill -9
